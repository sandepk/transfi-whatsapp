üîÑ Function Call Sequence
1
startUserRegistration(redisClient, whatsappNumber)
Initializes user registration state and returns welcome message
Parameters: redisClient, whatsappNumber
Returns: Welcome message string
2
setUserCreationState(redisClient, whatsappNumber, state)
Stores initial registration state in Redis with 1-hour expiration
Parameters: redisClient, whatsappNumber, state object
Returns: boolean (success/failure)
3
processUserRegistrationStep(redisClient, whatsappNumber, userInput)
Processes each user input during registration flow
Parameters: redisClient, whatsappNumber, userInput
Returns: Next question or completion message
4
getUserCreationState(redisClient, whatsappNumber)
Retrieves current registration state from Redis
Parameters: redisClient, whatsappNumber
Returns: state object or null
5
validateUserInput(userInput, validationType)
Validates user input based on field type (email, date, phone, etc.)
Parameters: userInput, validationType
Returns: {valid: boolean, message: string}
6
setUserCreationState(redisClient, whatsappNumber, updatedState)
Updates registration state with new collected data
Parameters: redisClient, whatsappNumber, updatedState
Returns: boolean (success/failure)
7
createUserAccount(state.collectedData)
Makes API call to Transfi to create user account
Parameters: collectedData object
Returns: Success message or error message
8
setUserCreationState(redisClient, whatsappNumber, null)
Clears registration state after completion
Parameters: redisClient, whatsappNumber, null
Returns: boolean (success/failure)
